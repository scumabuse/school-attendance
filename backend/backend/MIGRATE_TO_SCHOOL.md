# Инструкция по миграции на школьную систему

Этот скрипт переделывает систему посещаемости колледжа под школу.

## Что делает скрипт:

1. Создает классы: 1А, 1Б, 1В, 2А, 2Б, 2В... 11А, 11Б, 11В (всего 33 класса)
2. Распределяет всех студентов по классам случайным образом (25-30 учеников на класс)
3. Лишних студентов (если есть) оставляет в старых группах - они будут скрыты в интерфейсе

## Как запустить:

1. Убедитесь, что база данных подключена и доступна
2. Перейдите в директорию `backend/backend`
3. Запустите скрипт:

```bash
node migrate-to-school.js
```

## Важно:

- Скрипт не удаляет старые группы, только создает новые классы
- Студенты из старых групп будут скрыты в интерфейсе (фильтруются по паттерну классов)
- Если класс уже существует, он не будет пересоздан
- Все данные о посещаемости сохраняются

## После миграции:

- В интерфейсе будут отображаться только классы (1А-11В)
- Вкладка "Студенты" переименована в "Ученики"
- Все упоминания "Группа" заменены на "Класс"
- Старые группы скрыты, но данные не удалены
